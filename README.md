# 🍻 Choperia 820 - Delivery App

Um aplicativo completo de cardápio digital e delivery desenvolvido em Flutter. Criado para modernizar o atendimento e facilitar os pedidos da Choperia 820, localizada em Orlândia - SP.

## ✨ Funcionalidades

* **Cardápio Digital Interativo:** Navegação intuitiva por categorias (Couvert, Petiscos, Porções, Bebidas).
* **Carrinho de Compras:** Adição, remoção e controle de quantidade de itens em tempo real.
* **Gestão de Pedidos:** Acompanhamento do status do pedido desde a preparação até a entrega.
* **Painel do Cliente:** Histórico de pedidos e gerenciamento de perfil.
* **Integração em Tempo Real:** Conectado ao Firebase para envio instantâneo dos pedidos ao painel do restaurante.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas do desenvolvimento mobile:

* **[Flutter](https://flutter.dev/):** Framework principal para criação da interface multiplataforma.
* **[Dart](https://dart.dev/):** Linguagem de programação.
* **Provider:** Gerenciamento de estado da aplicação (Carrinho, Pedidos, Autenticação).
* **[Firebase](https://firebase.google.com/):** Banco de dados na nuvem (Firestore) para processamento em tempo real.
* **Device Preview:** Para testes rigorosos de interface (UI/UX) simulando diversos tamanhos de tela.
